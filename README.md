# AHB-APB-Bridge-Verilog
AHB-APB Bridge Design using Verilog
Overview:
This project implements an AHB to APB Bridge based on the AMBA protocol.

Modules:
AHB Slave Interface
APB Controller
AHB-APB Bridge Top Module

Features:
Supports AHB read/write transactions
APB protocol conversion
FSM-based APB Controller
Verified using ModelSim

Tools Used:
Verilog HDL
Quartus Prime
ModelSim
